دليل البدء السريع: كيفية استخدام أداة رسم الإحداثيات

  1. افتح الصفحة الرئيسية (index.html) وستظهر لك الخريطة.
  2. انقر على الخريطة لإضافة نقاط GPS. يمكنك إضافة أكثر من نقطتين لرسم مضلع.
  3. بعد إضافة 3 نقاط أو أكثر، انقر بزر الفأرة الأيمن على الخريطة لعرض قائمة منبثقة:
    • رسم مضلع وعرض البيانات: يرسم مضلعاً ويعرض المساحة والمحيط وعدد النقاط.
    • الحصول على جميع المسافات: يعرض المسافات بين كل نقطتين متتاليتين.
    • نسخ نقاط GPS إلى المدخل: ينسخ الإحداثيات إلى مربع الإدخال بالتنسيق المختار.
  4. يمكنك تعديل المضلع بسحب النقاط أو النقر عليه ثم الضغط على زر "تم" بعد الانتهاء.
  5. انقر بزر الفأرة الأيمن على أي مضلع لعرض قائمة:
    • نسخ إحداثيات GPS إلى المدخل
    • حذف المضلع
  6. نتائج الحسابات (المساحة، المحيط، عدد النقاط) والمسافات بين النقاط تظهر في البطاقات أعلى الصفحة.
  7. يمكنك تصدير أو استيراد البيانات، أو نسخ الإحداثيات، أو طباعة الخريطة من خلال الأزرار أسفل الصفحة.
  8. يمكنك اختيار اللغة (العربية أو الإنجليزية) من الأعلى لتغيير لغة الواجهة.

Quick Start Guide: How to Use the GPS Coordinate Plotter

  1. Open the main page (index.html) and the map will appear.
  2. Click on the map to add GPS points. You can add more than two points to draw a polygon.
  3. After adding 3 or more points, right-click on the map to show a popup menu:
    • Draw Polygon & Show Data: Draws a polygon and displays area, perimeter, and point count.
    • Get All Distances: Shows distances between each pair of consecutive points.
    • Copy GPS Points to Input: Copies coordinates to the input box in the selected format.
  4. You can edit the polygon by dragging points or clicking it, then press "Done" when finished.
  5. Right-click any polygon to show a menu:
    • Copy GPS Coordinates to Input
    • Delete Polygon
  6. Calculation results (area, perimeter, point count) and distances appear in the cards at the top of the page.
  7. You can export/import data, copy coordinates, or print the map using the buttons at the bottom of the page.
  8. You can select the language (Arabic or English) from the top to change the interface language.

Coordinate Format

Supported Coordinate Systems

The application supports two coordinate systems:

UTM System

  • Format: Easting,Northing
  • Example: 123456.78,9876543.21
  • UTM Zone: 38N
  • Suitable for agricultural surveying

Lat/Lon System

  • Format: Latitude,Longitude
  • Example: 31.9539,35.9106
  • WGS84 system
  • Suitable for general use

Supported Formats

The application accepts coordinates in the following formats:

# Single point format Easting,Northing (UTM) 123456.78,9876543.21 Latitude,Longitude (Lat/Lon) 31.9539,35.9106 # Multiple points format (one point per line) 123456.78,9876543.21 234567.89,9876544.32 345678.90,9876545.43 # Multiple points with separator * 123456.78,9876543.21*234567.89,9876544.32 345678.90,9876545.43*456789.01,9876546.54

Important Notes

  • Choose the appropriate coordinate system before entering data
  • Minimum 3 points required to draw a polygon
  • Use comma (,) to separate coordinates
  • Use * to separate points on the same line
  • Multiple lines can be entered for multiple points
  • Coordinates will be automatically converted for map display
  • Polygons can be edited after drawing by dragging numbered points
  • Double-click on a numbered point to add a new point
  • Double-click on a numbered point to remove it (if more than 3 points)
  • All changes are saved automatically in the browser
